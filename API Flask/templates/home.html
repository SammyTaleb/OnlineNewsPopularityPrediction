Machine Learning API



In this API, I have deployed two Machine Learning Classifiers to make predictions on the dataset called Online News Popularity. The dataset Online News Popularity takes 55 features to predict wether an article will be popular or not. The popularity depends on the number of time an article will be shared. A number of shares higher than 1 400 means that the article is a popular one. The features used to make such a prediction come from the analysis of the article. There are some general features (such as number of links, pictures or videos) and some features found using NLP on the article.
To make some predictions, you can select your classifier in the prediction tab and use the form to fill in the values for each variable. The prediction will be displayed after pressing the 'predict' button.
